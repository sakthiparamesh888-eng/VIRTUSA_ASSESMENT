Description

FareCalc is a simple ride fare calculator for a ride-sharing service.
The program estimates the cost of a ride based on:

Distance traveled
Type of vehicle selected
Time of the ride

During peak hours, surge pricing is applied automatically.

Features
Vehicle rate storage using a dictionary
Function for fare calculation
Surge pricing during peak hours (5 PM – 8 PM)
Input validation for distance, vehicle type, and time
Displays a detailed ride estimate
Technologies Used
Python
Dictionary data structure
Functions
Conditional statements
Error handling
